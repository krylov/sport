1. Examples of using odds.py tool

1.1. Load odds of season.

odds.py --season-url=http://www.oddsportal.com/soccer/spain/primera-division/results/
odds.py --season-url=http://www.oddsportal.com/soccer/england/premier-league/results/
odds.py --season-url=http://www.oddsportal.com/soccer/italy/serie-a/results/
odds.py --season-url=http://www.oddsportal.com/soccer/france/ligue-1/results/
odds.py --season-url=http://www.oddsportal.com/soccer/germany/bundesliga/results/

1.2. Load odds of current season.

odds.py --season-url=http://www.oddsportal.com/soccer/england/premier-league/results/ --add-season-years=2015-2016
